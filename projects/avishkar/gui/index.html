Avishkar GUI

Base Station Dashboard for Hyperloop Pod

React Electron MQTT WebSockets

This Graphical User Interface (GUI) serves as the mission control for the Avishkar Hyperloop pod. It communicates with the pod's onboard systems via MQTT over WebSockets, allowing for real-time telemetry monitoring and command execution. Key features include live plotting of sensor data, battery management system (BMS) monitoring, levitation gap sensing, and propulsion control.

Main Dashboard: Overview of critical systems including position, speed, and sensor health.

Propulsion Tab: Detailed view of inverter status, motor currents, and temperature monitoring.

Levitation Tab: Real-time air gap monitoring and stability controls.

Battery Tab: Individual cell voltage monitoring and thermal management for the high-voltage pack.
