Avishkar GUI

Base Station Dashboard for Hyperloop Pod

React Electron MQTT WebSockets

I led the development of the graphical user interface (GUI) that served as the base station for the Avishkar Hyperloop pod. It was build using React-Bootstrap for the frontend and a Node.js backend, leveraging Electron to package it as a desktop application. It communicates with the pod's onboard systems via MQTT over WebSockets, allowing for real-time telemetry monitoring and command execution. Key features include live plotting of sensor data, battery management system (BMS) monitoring, levitation gap sensing, and propulsion control.

1 / 4
Main Dashboard: Overview of critical systems including position, speed, and sensor health.
2 / 4
Propulsion Tab: Detailed view of inverter status, motor currents, and temperature monitoring.
3 / 4
Levitation Tab: Real-time air gap monitoring and stability controls.
4 / 4
Battery Tab: Individual cell voltage monitoring and thermal management for the high-voltage pack.
