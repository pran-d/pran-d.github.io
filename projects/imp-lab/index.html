Intelligent Machine Programming Lab

Drawing on a whiteboard with the UR10e and RoboDK

Robotics UR10e Python RoboDK Motion Planning

This project implements a robotic drawing system that can draw circles and SVG files onto a whiteboard using the UR10e robotic arm and RoboDK simulator/programming software. It handles scaling, positioning, and orientation of the tool relative to a user-defined drawing frame.

Key Features

  • Read and parse SVG files using svgpathtools
  • Draw precise circular trajectories with configurable parameters
  • Convert SVG to optimized robot trajectories
  • Scaling and Positioning of the drawing based on user input
  • Customizable drawing frame and tool orientation
  • Support for both local and URL-based SVG files
  • Real-time path visualization and simulation in the RoboDK sim environment

Authors: Ege Palamutlu, Kanish Thiagarajan, Pranav Debbad
