Intelligent Machine Programming Lab

Drawing and erasing on a whiteboard with the UR10e and RoboDK

Motion Planning ROS2 UR10e Franka FR3 RoboDK

This project involved using a robotic system to draw and erase circles and SVG files on a whiteboard using different robotic systems. The assignment involved two separate tasks:
Drawing SVG files and circles (with a user-defined center and radius) on a whiteboard using the UR10e arm, RoboDK (a simulation & programming software) and Python.
Erasing a whiteboard using the Franka FR3 arm, C++, ROS2 and the libfranka library.

Drawing task:

  • Setting up the UR10e robot and RoboDK simulation environment
  • Calibrating the tool (marker) frame using the UR10e teach pendant
  • Calibrating the whiteboard frame using the teach pendant
  • Setting up a simulation environment on RoboDK with accurate relative positioning of robot and whiteboard
  • Implementing a Python script to plan robot trajectories based on an input SVG file
  • Safely deploying the program on hardware, defining approach, drawing and retract phases

Erasing task:

  • Setting up the Franka FR3 robot and ROS2 environment with the libfranka library
  • Implementing a C++ program to move the robot manually in free-drive and record the motions without the eraser
  • Developing an impedance controller using ROS2 to playback the recorded motions while gripping the eraser
